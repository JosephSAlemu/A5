# Project Overview

A Java-based application made to be a simple Amazon clone.

Features:
- Purchasing electronics and other items
- Shopping carts

The project also uses Maven and a GitHub workflows to perform automatic testing and continuous integration.

Also uses:
- Static analysis using Checkstyle
- JUnit tests (Integration and Unit tests)
- Coverage report using Jacoco
